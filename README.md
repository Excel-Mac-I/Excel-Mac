## Overview

Excel Professional Suite is a complete data analysis platform with Power Query, Power Pivot, and collaboration for professional data workflows.

Power Query imports, transforms, and cleans data from dozens of sources with a visual transformation editor that handles the data preparation that analysis requires.

Power Pivot creates data models with relationships between tables, calculated columns, and DAX measures for advanced analysis beyond standard pivot tables.

Collaboration through 365 enables simultaneous multi-user editing with real-time updates and version history.

Excel Professional Suite is the data platform for professionals who need comprehensive spreadsheet capability with the data transformation, modeling, and collaboration tools that professional data analysis demands.

## Performance & Integration

Excel is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Excel processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
